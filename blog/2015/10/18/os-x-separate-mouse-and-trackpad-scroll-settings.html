Use Karabiner (https://github.com/tekezo/Karabiner) to change mouse and trackpad scroll settings. Edit private.xml.

SOMEVENDOR 0x0001 SOMEPRODUCT 0x0002 custom custom DeviceVendor::SOMEVENDOR, DeviceProduct::SOMEPRODUCT __FlipScrollWheel__ Option::FLIPSCROLLWHEEL_VERTICAL

Replace vendor id 0x0001 and product id 0x0002 with the values from EventViewer. Save private.xml and click the Reload XML button under the Change Key tab .

Open EventViewer.app using

$ open /Applications/Karabiner.app/Contents/Applications/EventViewer.app
