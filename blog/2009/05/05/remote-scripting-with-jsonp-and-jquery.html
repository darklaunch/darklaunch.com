Using JSONP allows for cross-domain HTTP GET requests. The following code uses jQuery to call the Flickr API for a list of photos returned in JSON.

 $.ajax({ type: 'GET', url: 'http://api.flickr.com/services/feeds/photos_public.gne?format=json&jsoncallback=?', dataType: 'jsonp', success: function(data){ $.each(data.items, function(i, item){ $('').attr('src', item.media.m).appendTo('#photos'); }); } }); NOTE: Updating is also possible using a GET request, but limited to around 2000 characters for some browsers and not recommended.
