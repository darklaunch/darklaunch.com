1. Download and install 7-zip (free) from http://www.7-zip.org/download.html 2. Open My Computer/Windows Explorer. Select Tools> Folder Options. 3. Select the File Types tab and click the New button. 4. Type zip into the file extension text field and click OK. 5. Select the ZIP extension if not already selected and click change. 6. Click Select the program from a list and click OK. 7. Browse to C:\Program Files\7-Zip\7zFM.exe and click Open. 8. Click OK and zip files will now be opened with 7-zip. Also, FileTypesMan from NirSoft is an excellent file types manager: http://www.nirsoft.net/utils/file_types_manager.html
