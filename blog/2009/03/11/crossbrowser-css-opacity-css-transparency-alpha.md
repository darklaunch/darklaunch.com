Transparency using CSS:

```css
.transparent {
    -khtml-opacity: .5;
    -moz-opacity: .5;
    -ms-filter: 'alpha(opacity=50)';
    filter: alpha(opacity=50);
    opacity: .5;
}
```

> NOTE: Please use sass.
