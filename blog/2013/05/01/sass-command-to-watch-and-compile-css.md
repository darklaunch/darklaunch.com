Write manageable CSS using Sass / SCSS. Compile to CSS with these options:

```sh
$ sass --watch style.scss:style.css --style expanded
```

```sh
$ sass \
    --watch style.scss:style.css
    --style expanded
```

Learn more at https://sass-lang.com/
