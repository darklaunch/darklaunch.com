How to fix:

Edit quoteCommand() in ../include/command.php quoteCommand() should read: function quoteCommand($cmd) { global $config; // On Windows machines, the whole line needs quotes round it so that it's // passed to cmd.exe correctly if ($config->serverIsWindows) { $cmd = '"'.$cmd.'"'; } else { $cmd = 'sudo -u root ' . $cmd; } return $cmd; } Add: else { $cmd = 'sudo -u root ' . $cmd; } to give svn permission.
